Игра кубики.
Нужно убирать блоки кубиков, соединенные вместе.
Очки начисляются нелинейно.
Справа показывается кол-во очков за данный блок и 
уже заработанные очки.


Управление:
1) Навести курсор на поле -
   подсвечивание выбранного блока кубиков.
2) ПКМ - убрать выделенный блок кубиков.
3) S - сохранить игру
4) L - загрузить предыдущее сохранение

Консольная версия:
Все аргументы и ключи не обязательные.
1) -h, --help : помощь
2) -s, --size : размер поя(small, middle, large, default - small)
3) -c, --colors : количество цветов(2, 3, 4, default - 2)
4) -r, --records : вывести таблицу рекордов
5) -n, --name : имя игрока(default - DefaultName)
